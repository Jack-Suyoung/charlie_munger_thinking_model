## 이 사이트는

찰리 멍거가 평생에 걸쳐 구축한 **90개 핵심 멘탈 모델**을 체계적으로 학습할 수 있도록 정리한 자료입니다.

## 구성

| 섹션 | 설명 |
|------|------|
| [90개 멘탈 모델](models.md) | Tier 1 (47개) + Tier 2 (43개) 전체 목록과 상세 설명 |
| [마스터 체크리스트](checklist.md) | 의사결정 전 점검용 90개 질문 체크리스트 |
| [세션별 학습](sessions/index.md) | 22개 세션으로 나눈 음성 학습 가이드 |

## 학습 로드맵

```
심리학 (세션 01-07) → 수학/통계 (08-10) → 자연과학 (11-14) → 경제학 (15-18) → 사고 도구 (19-22)
```

### 빠른 시작

1. [세션 1: 인센티브의 힘](sessions/session_01.md)부터 순서대로 학습
2. 각 세션 학습 후 [마스터 체크리스트](checklist.md)에서 해당 모델 복습
3. 전체 모델 참조가 필요하면 [90개 멘탈 모델](models.md) 페이지 활용
