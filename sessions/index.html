    사용법: ChatGPT 음성 대화¶

    방법 A: 매 세션 직접 입력 (간단)¶

    1. ChatGPT 앱을 열고 새 대화를 시작합니다
    2. system_prompt.md 안의 프롬프트를 복사해서 첫 메시지로 붙여넣습니다
    3. 바로 이어서 오늘 학습할 세션 파일 내용을 붙여넣습니다
    4. "시작해줘"라고 입력합니다
    5. 헤드폰 아이콘을 눌러 음성 모드로 전환합니다
    6. 대화하며 학습합니다

    방법 B: 커스텀 지침 활용 (추천)¶

    1. ChatGPT 설정 > 맞춤 지침(Custom Instructions) 으로 이동합니다
    2. system_prompt.md 안의 프롬프트를 "응답 방식" 칸에 저장합니다
    3. 이후 매 세션마다 세션 파일 내용만 붙여넣으면 됩니다
    4. 음성 모드로 전환하여 학습합니다

    방법 C: GPTs 활용 (고급)¶

    1. ChatGPT에서 나만의 GPT를 생성합니다
    2. 시스템 프롬프트를 GPT의 지침에 입력합니다
    3. 22개 세션 파일을 모두 Knowledge로 업로드합니다
    4. "세션 1부터 시작해줘"라고 말하면 순서대로 진행됩니다

    사용법: Gemini Live¶

    1. Google Gemini 앱을 엽니다
    2. 시스템 프롬프트 + 세션 내용을 텍스트로 입력합니다
    3. Live 버튼을 눌러 음성 모드로 전환합니다
    4. 대화하며 학습합니다

    음성 대화 팁¶

    • "다음" — 다음 모델로 넘어갑니다
    • "예시 더" — 추가 사례를 요청합니다
    • "복습" — 지금까지 배운 내용을 정리합니다
    • "이전 세션이랑 연결해줘" — 세션 간 연결고리를 설명합니다
    • "퀴즈 내줘" — 배운 내용을 테스트합니다
